# 💻 Portfólio — João Paulo | Dev

Site de portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, apresentando minha trajetória acadêmica, competências técnicas, projetos e certificações de forma interativa e visualmente moderna.

🔗 **Acesse o site:** [joaopaulomenezes01.github.io/portfolio](https://github.com/joaopaulomenezes01)

---

## 📋 Sobre o Projeto

Portfólio desenvolvido do zero com foco em performance e experiência visual. O projeto combina efeitos de partículas animadas, tilt 3D nos cards, efeito de digitação no título e animações de scroll, tudo sem nenhuma dependência de framework — apenas HTML, CSS e JavaScript nativos.

---

## ✨ Funcionalidades

- **Efeito de digitação** — Título animado letra por letra ao carregar a página
- **Partículas interativas** — Fundo animado com Canvas 2D, partículas conectadas por linhas dinâmicas
- **Cursor personalizado** — Cursor substituído por um ponto luminoso com `mix-blend-mode`
- **Tilt 3D nos cards** — Cards de projeto com rotação 3D responsiva ao movimento do mouse
- **Fade-in por scroll** — Cards de certificações e competências aparecem suavemente ao entrar na viewport (Intersection Observer)
- **Notificação de boas-vindas** — Toast animado ao carregar a página (inspirado no projeto PetShop)
- **Underline animado no nav** — Links da navegação com efeito de sublinhado deslizante ao hover
- **Scroll suave** — Navegação fluida entre seções
- **Layout responsivo** — Adaptado para dispositivos móveis

## 📄 Seções

### 🏠 Hero
Apresentação com badge de cargo, efeito de digitação, localização e botão de chamada para ação.

Texto de perfil profissional e quatro cards com informações de formação, idiomas, softwares e habilidades comportamentais.

Grid de competências em suporte técnico, redes, Linux, backup e manutenção de sistemas.

Grid com todas as certificações obtidas (Cisco, ENAP, SEST SENAT, IFPE), incluindo carga horária e notas quando disponíveis.

Ícones animados das tecnologias: Python, Java, JavaScript, HTML, CSS, MySQL, Spring e Linux.

### 📬 Contato
Links para LinkedIn, GitHub, e-mail e telefone com animação de deslizamento ao hover.

## ⚙️ Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/joaopaulomenezes01/portfolio.git
```

2. Acesse a pasta:
```bash
cd portfolio
```

3. Abra o `index.html` no seu navegador.

> Não é necessário instalar dependências. O projeto roda diretamente no navegador com conexão à internet para carregar as fontes e ícones via CDN.
